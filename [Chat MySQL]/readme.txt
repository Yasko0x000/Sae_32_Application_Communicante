En réalité nous avons nommée le premier dossier serveur et le deuxième client, cependant c'est faux en théorie 
puisque les 2 sont clients du serveur MySQL.


Pour que la connexion avec la bdd marche, il est neccessaire d'ajouter la librerie en .jar (mysql-connector-java-5.1.22-bin) 

Sur chaqu'un des projet chat MySQL, faire click droit dans Librairies > Add JAR/Folder > Selectionner le fichier .jar disponible dans ce dossier
